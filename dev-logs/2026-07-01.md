# 2026-07-01 开发日志

## 本日主题：T教练语音对话系统开发

---

## 一、语音对话核心功能

### 技术栈
全浏览器原生 API，零新增依赖，零 API 成本：
- STT：Web Speech API (SpeechRecognition)
- TTS：Web Speech API (SpeechSynthesis)
- 音效：Web Audio API 程序合成

### 组件架构

| 文件 | 功能 |
|:---|:---|
| `VoiceChat/useSpeechRecognition.ts` | STT 封装 — 权限、持续监听、自动重连、沉默兜底(2.5s) |
| `VoiceChat/useSpeechSynthesis.ts` | TTS 封装 — 朗读/音效、音色管理、朗读双模式 |
| `VoiceChat/commandParser.ts` | 本地指令解析 — 唤醒词剥离、3模式、9语言、兜底反馈 |
| `VoiceChat/VoiceChat.tsx` | 主组件 — 三模式Tab、消息流、设置面板、悬浮球、API Key管理 |
| `VoiceChat/VoiceChatPiP.tsx` | 画中画 — 透明叠加窗口、自动启动、穿透模式、跨窗口同步 |

### 三模式

| 模式 | 功能 |
|:---|:---|
| 🎓 T教练 | 语音提问 → AI 文字回答 + TTS 朗读 |
| 👂 听队友 | 外语识别 → 翻译成中文显示 |
| 🗣️ 我说话 | 中文 → 翻译成外语 + 自动复制到剪贴板 |

### 三种使用方式

1. Ctrl+Shift+T 打开内嵌面板
2. T教练下拉菜单「🎤 语音对话」
3. T教练下拉菜单「📌 画中画」— 独立透明窗口浮在游戏上方
