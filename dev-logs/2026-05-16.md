# 开发日志 — 2026-05-16（续）

## ValoPlant 功能复刻 — 全部完成

### 新增功能
- [x] **绘图工具系统** — ToolPalette 工具栏 + DrawingLayer SVG 绘制层
  - 9 种工具：选择/画笔/直线/箭头/矩形/圆形/文字/特工/橡皮
  - 7 种颜色 + 4 档线宽
  - 所有绘图跟随地图缩放/平移
- [x] **文字标注** — 文字工具在地图上添加可编辑文字
- [x] **特工位置标记** — 拖拽特工头像到地图放置攻防站位
- [x] **撤销/重做** — Ctrl+Z 撤销 / Ctrl+Y 重做，最多 50 步历史
- [x] **攻防切换** — 导航栏一键切换进攻/防守视角，地图 180° 旋转
- [x] **动画播放** — 时间轴播放/暂停/停止，速度 0.5x/1x/2x，逐步骤高亮
- [x] **策略描述** — 模板保存时添加描述字段，导航栏显示策略名
- [x] **移动端适配** — 767px 断点侧边栏折叠
