## 1. 快速开始

1. 打开网站后点击「进入战术板」或按 `Enter` 键
2. 在左侧**特工面板**中选择特工，展开后拖拽技能到地图上
3. 使用顶部工具栏切换地图、攻防方、绘图
4. 点击「保存进度」防止丢失，点击「导出」分享战术

## 3. 特工与技能

### 特工面板
- 左侧面板列出所有 29 名特工
- 搜索框支持**中/英文名**搜索
- 点击特工展开，显示 C / Q / E / X 四个技能
- 每个技能显示**名称、按键、类型标签**

### 技能图标颜色含义
| 颜色 | 类型 | 说明 |
|------|------|------|
| 🟢 绿色 | 烟雾 (Smoke) | 遮挡视野 |
| 🟡 黄色 | 闪光 (Flash) | 致盲效果 |
| 🔴 红色 | 伤害 (Damage) | 造成伤害 |
| 🔵 蓝色 | 侦查 (Recon) | 信息获取 |
| 🟣 紫色 | 控制 (Control) | 减速/压制 |
| 🟢 亮绿 | 治疗 (Heal) | 回复生命 |
| 🟠 橙色 | 位移 (Mobility) | 快速移动 |

### 放置技能
- 直接**拖拽技能**到地图上即可放置
- 圆形技能：放开即放置
- 线型/锥形技能：进入**画线模式**（见第6节）
- 点击技能旁的 `?` 按钮查看技能详细说明

## 4. 阵容构建

在特工面板顶部的**阵容区**可组建 5 人队伍：

- **拖拽特工头像**到「进攻方」或「防守方」槽位
- 每方最多 5 人
- **左键点击**槽位中的头像：展开该特工并滚动到对应位置
- **右键点击**槽位中的头像：移除

## 5. 绘图工具

工具栏位于画布左上方，提供以下模式：

| 工具 | 图标 | 用途 |
|------|------|------|
| 选择 | ✋ | 点击选中、拖拽移动地图元素 |
| 画笔 | ✏️ | 自由绘制线条 |
| 直线 | 📏 | 画直线段 |
| 箭头 | ➡️ | 画带箭头方向线 |
| 矩形 | ⬜ | 画矩形框 |
| 圆形 | ⭕ | 画圆形圈 |
| 文字 | T | 点击添加文字标注 |
| 橡皮 | 🧹 | 点击删除任意元素 |

### 绘图配置
- **颜色**：红/黄/绿/蓝/紫/橙/白 七色可选
- **线宽**：1-12px 滑块调节
- **字号**：14/18/24/32/48 五档可选（文字工具）
- 工具栏可**折叠**，只显示当前工具图标

### 文字标注
- 选择文字工具后点击地图
- 输入文字（最多50字）、选颜色和字号
- **双击**已有文字可编辑或删除
- **拖拽**文字可移动位置

## 6. 技能放置模式

### 直接放置（圆形技能）
拖拽圆形技能到地图上直接放置，显示圆形范围。

### 画线模式（线型/矩形/锥形技能）
拖拽后进入特殊交互：

| 模式 | 操作 | 示例 |
|------|------|------|
| **直线两点** | 点击起点 → 点击终点 | 猎枭X、欧门C |
| **固定长度直线** | 点击起点 → 终点自动锁定固定距离 | 铁臂E/X、黑梦X |
| **自由绘制** | 按住拖拽画线 → 松手确认 | 海神Q、不死鸟C、盖可X |
| **锥形拖拽** | 点击锥尖 → 点击确定方向 | 铁臂C、迷核X |
| **矩形拖拽** | 点击起点 → 终点决定方向 | 钛狐X |
| **多段线** | 多次点击添加节点 | 钢锁X |
| **钛狐Q / 幻棱Q** | **左键**=一段完成 / **右键**=两段模式 | |

> 画线模式中按 `ESC` 取消，预览虚线显示最终效果。

## 7. 时间轴与录制回放

### 录制
1. 点击时间轴顶部的 **⏺ 录制** 按钮开始
2. 每次放置技能自动记录为一步（含时间戳）
3. 再次点击停止录制，生成一条**轨道**

### 轨道管理
- 每条轨道显示名称、步数
- **双击轨道名**可重命名
- 点击轨道查看步骤列表
- 支持**删除**轨道

### 回放
- 选择轨道后点击 **▶ 播放**
- 技能按录制顺序逐步显示（每步约1.5秒间隔）
- 回放中地图只显示已展示的技能
- 可随时停止回放

## 8. 模板管理

点击导航栏**「模板管理」**按钮打开：

| 操作 | 说明 |
|------|------|
| 💾 保存当前 | 命名并保存当前战术到本地数据库 |
| 📂 加载模板 | 恢复之前保存的战术（含地图切换） |
| 📥 导出JSON | 下载 `.json` 文件，可分享给他人 |
| 📤 导入JSON | 加载他人分享的战术文件 |
| 🗑 清空画布 | 清除所有元素（需确认） |
| 💾 保存进度 | 自动保存到浏览器本地存储 |

## 9. 导出与分享

### 导出图片
导航栏点击「导出」→ 选择「导出图片」→ 自动下载 PNG 图片（1800×1200）

### 分享链接
导航栏点击「导出」→ 选择「分享链接」→ 生成带战术数据的 URL → 复制发送给队友

> 链接数据限制约 3000 字符，超长战术建议使用 JSON 导出。

### 导出 JSON
模板管理中导出 `.json` 文件，包含完整战术数据，可跨设备导入。

## 11. 技能图标模式

大量技能默认为**「仅图标」模式**，避免地图过于拥挤：

- 放置后只显示**技能图标**（18px）
- **点击图标**展开完整范围
- **取消选中**恢复仅图标
- 右下角 **👁‍🗨 按钮**一键显示/隐藏所有范围
- 线型技能图标显示在**终点**，矩形技能在**起点**

### 钢锁C特殊操作
- 选中后四臂末端出现**白色手柄**
- 上下/左右拖拽独立调节每根线的长短
