# 无畏契约经济系统详解

## 一、金币收入

| 收入来源 | 金额 | 备注 |
|----------|------|------|
| 手枪局初始 | 800 | 每个半场第一回合固定 |
| 击杀敌人 | 200/人 | 任意方式击杀均为200 |
| 回合胜利 | 3000 | 全队胜者统一获得 |
| 回合失败 | 1900 | 首败基础金额 |
| 下包（安放爆能器） | 300/人 | 进攻方全队每人+300 |
| 拆包 | 300 | 仅最终拆包者获得 |
| 保枪（存活） | 1000 | 输掉回合但存活时（有例外） |
| 加时赛初始 | 5000 | 加时阶段固定资金 |
| 账户上限 | 9000 | 多余时给队友发枪 |

## 二、连败补偿机制

| 连败次数 | 失败收入 | 说明 |
|----------|----------|------|
| 首次失败 | 1900 | 无补偿基础值 |
| 连败第1次 | 2400 | +500补偿 |
| 连败第2次 | 2900 | +1000补偿（封顶） |
| 连败第3次起 | 2900 | 封顶不再增加，直到获胜重置 |

**关键规则：**
- 连败一方赢下任一回合后，再次输直接从1900重新计算
- 防守方没拆包或进攻方没下包就输了 → 保枪存活者固定只拿1000（不管连败几局）
- 但如果进攻方是爆能器被拆除导致输 → 保枪队员仍按连败补偿正常拿钱

## 三、关键经济门槛

| 配置 | 价格 | 构成 |
|------|------|------|
| 半甲+主战步枪 | 3300 | 轻甲400 + 狂徒/幻影2900 |
| 全甲+主战步枪 | 3900 | 重甲1000 + 狂徒/幻影2900 |
| 全技能价格 | 约600-800 | 捷风最贵≈800，钱博尔猎头子弹150/颗 |

**完整长枪局理想配置**：重甲1000 + 步枪2900 + 全技能≈700 = **约4600**

## 四、经济局类型

| 类型 | 花费 | 策略 |
|------|------|------|
| **ECO局** | 0-1200 | 存钱，不起大枪大甲，只为下局 |
| **半起局** | 1500-3000 | 起便宜枪+半甲，博翻盘 |
| **强起局** | 花光所有 | 全力一搏，翻盘则对方经济炸 |
| **全起/长枪局** | 3300-4600 | 主战枪+甲+技能全齐 |
| **奖励局** | 保留节省 | 手枪局赢后第三局选择省经济 |

## 五、经典经济路线

### 赢了手枪局
- 第二局 → 全甲+冲锋枪/便宜枪（确保拿下）
- 第三局 → 可打"奖励局"保留经济，或补枪

### 输了手枪局
- 第二局 → 建议什么都别买（纯ECO），攒钱
- 第三局 → 3000(输)+2400(连败)=5400，够全甲+步枪3900+技能
- 此时对面可能还在用第二局弱枪，枪械反而优势

## 六、经济决策要点

1. **同步经济**：全员策略尽量一致，队友起你才起
2. **半甲vs全甲**：决定"两枪死还是三枪死"
3. **技能不消失**：ECO局可提前买技能储备（不会因阵亡消失）
4. **发枪给队友**：账户快满9000时
5. **进攻方输局尽量下包**：避免被判定保枪只拿1000
6. **保枪时机**：队友全死、1打多、敌人已下包→立刻躲起来保枪
7. **缴枪价值**：胜局已定→集体搜点打掉保枪敌人，摧毁对方经济
