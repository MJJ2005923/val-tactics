## 十三、阵容搭配原则

| 原则 | 说明 |
|------|------|
| **至少1烟** | 无烟进点极度困难，烟雾是进攻防守核心 |
| **至少1信息** | 先锋提供侦查，没有信息的队伍如瞎子 |
| **至少1哨卫** | 防守需要陷阱/摄像头看后路，进攻需要断后 |
| **双烟适用** | 蝰蛇+任意烟，适合需要多角度分割的地图 |
| **双先锋适用** | 地图死角多、开阔需要大量信息时 |
| **双决斗适用** | 强调进攻速度和火力压制时 |

### 2026赛季 Meta 趋势

- 双先锋越来越主流（信息→击杀转化率提升）
- 蝰蛇在部分地图仍是不可或缺的双烟核心
- 哨卫中奇乐出场率最高，零紧随其后
- 捷风/雷兹仍是决斗者首选，幻棱和夜露快速崛起
