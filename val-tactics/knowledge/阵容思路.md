## 十三、阵容搭配原则

| 原则 | 说明 |
|------|------|
| **至少1烟** | 无烟进点极度困难，烟雾是进攻防守核心 |
| **至少1信息** | 先锋提供侦查，没有信息的队伍如瞎子 |
| **至少1哨卫** | 防守需要陷阱/摄像头看后路，进攻需要断后 |
| **双烟适用** | 蝰蛇+任意烟，适合需要多角度分割的地图 |
| **双先锋适用** | 地图死角多、开阔需要大量信息时 |
| **双决斗适用** | 强调进攻速度和火力压制时 |

### 2026赛季 Meta 趋势（高段位 + VCT）

- **双先锋仍是主流**：猎枭+KAYO 信息网在高段位几乎必备，信息→击杀转化率持续提升
- **暮蝶称霸天梯**：全图最高胜率（52-53%），死后封烟+杀人回血+大招复活，单排首选控场
- **VCT 霓虹统治**：职业赛场 62% 出场率，但天梯受 12.09 削弱影响降至 A/B 级
- **幻棱跌落神坛**：Kickoff 42.7% → Stage 1 仅 4-13%，被 12.09 大削后高段位极少出场
- **贤者重回 S 级**：冰墙+复活在配合好的对局价值极高，Split/Haven/Bind 表现最佳
- **婕提/雷兹仍是决斗核心**：高段位和职业赛场双雄，机动性和爆发力不可替代
- **蝰蛇+幽影双烟**：VCT 最稳定控场组合，部分地图蝰蛇仍不可或缺
