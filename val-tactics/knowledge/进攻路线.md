# 无畏契约各图进攻路线（2026赛季）

## 一、亚海悬城 (Ascent)

### A点标准爆弹
1. **控中路**：开局先锋抢占中路控制权（中门/猫道），烟位封中门上
2. **分割防守**：烟位封CT和二楼上，断掉防守方回A的路线
3. **A大推进**：3人从A大厅推到A大，清理树位
4. **A小同步**：2人从中路→A小拉出，与A大形成夹击
5. **闪光进点**：A大扔瞬爆闪清二楼下和死点，A小闪光清二楼上
6. **下包位置**：A点默认位置（靠A大一侧），守包时A大和A小各站1人

### B点标准爆弹
1. **控中路+市场**：先锋+哨卫静步摸市场，烟位封中门下
2. **封CT+市场门**：烟位封CT烟和市场门烟
3. **B大推进**：3人从B大主攻，清船屋
4. **市场夹击**：2人从市场门同时拉出
5. **清死角**：B小和船屋用狗/无人机确认
6. **下包位置**：B点默认（靠B大一侧），守包分散B大和船屋

### 快捷路线
- **中路→猫道→A小**：绕过A大正面防守，直接威胁A包点
- **市场→B小**：中路控制后快速转B，打防守方措手不及

## 二、源工重镇 (Bind)

### A点标准爆弹
1. **封CT烟**：炼狱/欧门封A点CT入口烟
2. **A大推进**：4人从A大主攻，清浴室
3. **A小利用传送门**：1人走传送门绕到A小（从B大TP到A小浴室）
4. **同步拉出**：A大和浴室TP口同时拉
5. **下包位置**：A点靠A大一侧，守包用燃烧弹覆盖炸弹

### B点标准爆弹
1. **封CT+水烟**：烟位封B点CT烟和水烟窗口烟
2. **B大推进**：主力从B大推进，清花园
3. **传送门绕后**：1人从A小TP传送到B大TP口，双面包夹
4. **闪光开路**：瞬爆闪清B后点和CT位置
5. **下包位置**：B点默认，守包时花园和B后各站1人

### 假打策略
- A点扔烟+火假装爆弹→防守方从B转点→TP快速传到B打空点
- 绑定的双TP是该图最大特色，假打效率全游戏最高

## 三、隐世修所 (Haven)

### A点标准爆弹
1. **封CT+二楼上**：烟位封A点CT烟和二楼上烟
2. **A大推进**：3人从A大厅推A大，先锋用狗/无人机清角落
3. **A小同步**：2人从中路连接→A小拉出
4. **闪光清二楼上**：不死鸟/斯凯瞬爆闪清二楼上
5. **下包位置**：A点靠A后一侧，守包A大+A后交叉

### C点标准爆弹
1. **封CT+车库**：烟位封C点CT烟和车库烟
2. **C大推进**：4人从C大厅推C大
3. **C小压力**：1人在C小牵制，不让防守方从C小反清
4. **下包位置**：C点默认，守包C后平台+C大

### B点（中路）快速快攻
1. 最快突破：5人开局从进攻方中路直冲B点
2. 只需要封防守方中路看台烟+CT烟
3. 速度压制，手枪局/ECO局屡试不爽

---

## 四、霓虹町 (Split)

### A点标准爆弹
1. **控中路**：先锋抢占中路上层，烟位封中路下层→收发室
2. **封CT+二楼上**：烟位封A点CT烟和Heaven烟
3. **A大推进**：3人从A大厅攻A大
4. **A小（中路上层）同步**：2人从中路上层→A包点
5. **清屏幕+A斜坡**：用闪光逼出掩体后敌人
6. **下包位置**：A斜坡一侧，守包A大+Heaven交叉

### B点标准爆弹
1. **封CT+B二楼上**：烟位封B点CT烟和Heaven烟
2. **B大推进**：4人从B大厅攻B大，清车库
3. **中路下层配合**：1人从通风口→B二楼上（抄后路）
4. **闪光清B二楼下**：瞬爆闪清掩体后
5. **下包位置**：B点靠B大一侧，守包B二楼上+B大

## 通用进攻法则

1. **先控制后执行**：先确保中路/侧翼控制权，再发动总攻
2. **烟到位才拉**：烟雾弹确认展开后才能拉出
3. **信息先行**：无人机/狗/侦查箭 → 确认位置 → 技能打击 → 进点
4. **同步大于一切**：双路夹击必须秒级同步，一前一后等于送人头
5. **下包后立刻散开**：包点内最多1人，其余分散外围交叉架枪
6. **变奏是关键**：默认→慢打→快攻→假打→爆弹，混着用让防守方无法预判
