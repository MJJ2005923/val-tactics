# 后续计划：AI 战术助手

> 状态：规划中 | 预计总工期：3-4 周 | 起步成本：¥0

---

## 概述

把无畏契约战术板升级为一个 **AI 驱动的战术教学平台**，分三步走：先在网站验证，再做可安装版，最后上架独立 App。

---

## 三个阶段

### 第一步：网站接入 AI（1-2 周，¥0）

**目标：** 在现有的 `val-tactics.pages.dev` 上加 AI 功能，用户自带 API Key，我们收集匿名数据。

**技术架构：**

```
用户浏览器                  Cloudflare Worker              AI 平台
┌──────────┐   POST/api     ┌──────────┐   转发+Key       ┌──────────┐
│ 前端页面  │ ────────────→ │ 代理中转  │ ───────────────→ │ Anthropic │
│          │               │          │                  │ OpenAI    │
│ Key 存   │ ←──────────── │ 无状态   │ ←─────────────── │ Google    │
│ localStorage             │ 不存 Key │                  │ DeepSeek  │
└──────────┘               └────┬─────┘                  └──────────┘
                                │
                                │ 匿名储存
                                ▼
                         ┌──────────┐
                         │ D1/KV    │
                         │ 问答记录  │
                         │ 用户评分  │
                         └──────────┘
```

**要做的事：**

| 任务 | 内容 |
|------|------|
| 设置页面 | API Key 输入 + 模型选择，数据存 localStorage |
| Worker 代理 | 纯转发 API 请求到四大平台，不存 Key |
| 知识库 | 你整理的战术数据 JSON → KV，AI 回答前先检索 |
| 数据采集 | 匿名问答对 + 👍👎评分 → D1，喂知识库 |
| AI 对话面板 | 侧边栏聊天窗口，上下文含当前地图+技能布局 |

**运营成本：¥0/月。** Worker + KV + D1 全部免费额度。

**数据采集策略：**

| 采什么 | 用途 | 隐私 |
|------|------|------|
| 匿名问答对 | 训练知识库 | 🟢 安全 |
| 👍/👎 评分 | 质量排序 | 🟢 安全 |
| 热门问题 | 发现需求 | 🟢 安全 |
| API Key | ❌ 不存 | — |

---

### 第二步：PWA 化（1-2 天，¥0）

把现有网站变成可安装的"伪 App"，添加到手机桌面即可用：

- 全屏运行无地址栏
- 离线缓存
- 推送通知
- 不需要应用商店

加一个 `manifest.json` + Service Worker 即可。

---

### 第三步：独立 App——"瓦搭子"（1-2 周，¥25 起步）

当网站 AI 验证成功后，用 Capacitor 把代码打包成独立 App。

**为什么安全：** 纯文字 AI，不碰拳头 IP（不用特工头像、技能图标、地图纹理）。App 名叫「瓦搭子」或「瓦教练」，不叫无畏契约。

| 项目 | 费用 |
|------|------|
| Capacitor 打包 | ¥0 |
| Android 上架 | ¥25 一次性 |
| iOS 上架 | ¥688/年 |
| 运营成本 | ¥0/月 |
| **起步总成本** | **¥25** |

**功能：** 阵容推荐、战术教学、个人教练、对局复盘、版本解读——全部纯文字 AI。

## 远期规划（网站做起来后考虑）

### 地图 3D 化
- 伪 3D：CSS perspective 倾斜视角 + 透视感（1天）
- 简 3D：Three.js + 可旋转视角 + 墙体高度（3-5天）

- 难点：Riot 不公开 3D 模型、3D 技能放置 UX 复杂、WebGL 性能
