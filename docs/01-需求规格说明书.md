# 01 — 需求规格说明书

## 项目背景

无畏契约（VALORANT）是一款 5v5 战术射击游戏。玩家在游戏内无法方便地规划全队技能配合和战术路线。本工具提供一个网页端战术板，允许玩家在地图上拖拽摆放技能标记、编排释放顺序，并保存/分享战术模板。

## 目标用户

- 无畏契约玩家（个人使用）
- 战队和开黑小队（团队协作）
- 教练/分析师（战术讲解）

## 功能需求（MoSCoW 优先级）

### P0 — 必须有（MVP）

| 功能 | 描述 |
|------|------|
| 地图浏览 | 查看所有游戏地图，支持缩放和拖拽平移 |
| 技能库 | 全部英雄和技能的详细信息 |
| 拖拽部署 | 从侧边栏拖拽技能图标到地图上 |
| 步骤编号 | 已放置的技能标记显示序号 |
| 模板保存 | 战术模板保存到浏览器本地，支持多套 |
| JSON 导入导出 | 导出模板文件分享给队友，支持导入 |

### P1 — 应该有

| 功能 | 描述 |
|------|------|
| 时间轴 | 底部时间轴可视化技能释放顺序和时间间隔 |
| 标记备注 | 每个标记可添加文字说明 |
| 模板管理 | 重命名、删除、复制模板 |

### P2 — 可以有

| 功能 | 描述 |
|------|------|
| 键盘快捷键 | Ctrl+S 保存、Delete 删除标记 |
| 英雄搜索 | 侧边栏搜索/筛选英雄 |
| 响应式适配 | 支持不同屏幕宽度 |

### P3 — 未来考虑

| 功能 | 描述 |
|------|------|
| QQ/微信登录 | 账号系统 + 云端同步 |
| 实时协作 | 多人同时编辑 |
| 战术动画播放 | 按时间轴自动播放技能释放动画 |

## 非功能需求

| 类别 | 要求 |
|------|------|
| 浏览器兼容 | Chrome 90+、Edge 90+、Firefox 90+ |
| 屏幕分辨率 | 1366×768 及以上 |
| 性能 | 地图缩放帧率 ≥ 30fps |
| 数据安全 | 纯本地存储，不上传数据 |
| 包体积 | 首屏加载 ≤ 2MB（不含地图图片） |
