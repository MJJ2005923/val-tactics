## 1. 快速上手（5秒开始）

1. 打开网站 → 看到启动画面 → 点击 **「进入战术板」** 或直接按键盘上的 `Enter` 键。
2. 你就看到一张天漠之峽的地图了。在左边找到你想用的特工，点开，把技能图标拖到地图上。
3. 搞定！你已经放好了第一个战术标记。

> 💡 **提示**：如果你只想问 AI 战术问题，不需要画图。在战术板页面顶部点 **「T教练」** 就可以和 AI 对话。

## 2. 战术板 — 绘制你的战术

### 2.1 地图操作

- **换地图**：顶部工具栏有地图选择器，点击切换。
- **缩放地图**：用鼠标滚轮（触控板双指捏合）放大缩小。右下角有 `+` `-` 按钮也行。
- **移动地图**：按住鼠标拖拽即可平移。或者用右下角的 ⬆⬇⬅➡ 方向键。
- **切换攻防视角**：顶部选择「进攻方」或「防守方」，地图会对应旋转。

### 2.2 放置技能

左边是**特工列表**，列出了 29 位特工。操作步骤：

1. **找到特工**：可以滚轮翻找，也可以在顶部的搜索框直接打名字（支持中文和英文）。
2. **点开特工**：点击特工名字展开，你会看到四个技能按钮（C / Q / E / X）。
3. **拖到地图**：按住技能图标，拖到地图上你想放的位置，松开手就放好了。
4. **调整位置**：放好后可以拖拽技能圈移动它，也可以旋转（圆形技能顶部有个小圆点，拖它就能转）。

> 🎨 **技能颜色说明**：绿色=烟、黄色=闪、红色=伤害、蓝色=侦查、紫色=控制、亮绿=治疗、橙色=位移。

### 2.3 组阵容

在特工面板顶部有个阵容区：

- 拖拽特工头像到「进攻方」或「防守方」的格子里。
- 每方最多 5 个。左键点已放好的头像可以快速跳转到该特工，右键移除。
- 如果你在 T教练 那里提问，AI 会知道你组了什么阵容，给出的建议更精准。

### 2.4 画图标注

左上角的工具栏提供画图功能。很简单：

| 想做什么 | 怎么操作 |
|:---|:---|
| 画箭头 | 选「箭头」，在地图上按住拖拽 |
| 画方块 | 选「矩形」，按住拖拽框出范围 |
| 画圈 | 选「圆形」，按住拖拽 |
| 写文字 | 选「文字」，点地图，输入内容。双击已有文字可以改 |
| 随便画 | 选「画笔」，按住鼠标自由画 |
| 擦掉 | 选「橡皮」，点一下要删的东西就行 |
| 移动元素 | 选「选择」（手形图标），拖拽任意元素 |

画之前可以调颜色、线条粗细、文字大小。

### 2.5 比赛数据

如果你想分析自己的比赛：

- 在「数据分析」页面录入你的比赛记录（击杀、死亡、助攻、经济等）。
- 回到战术板，使用 **「数据分析」** 功能，T教练 就能结合你的真实对局数据给你建议。

### 2.6 保存和分享

- **自动保存**：你的操作会自动保存在浏览器里。关了网页再打开，所有东西都还在。
- **导出图片**：顶部「导出」→「导出图片」，下载一张 PNG 截图发给队友。
- **分享链接**：顶部「导出」→「分享链接」，复制链接发给队友，他们点开就能看到你的战术布置（需要同一浏览器暂不支持跨设备）。
- **模板管理**：顶部「模板管理」可以保存多个战术来回切换，也可以导出成 JSON 文件永久保存。

### 2.7 快捷操作

| 按键 | 做什么 |
|:---|:---|
| `Delete` 或 `Backspace` | 删除选中的东西 |
| `Ctrl + Z` | 撤销上一步 |
| `Ctrl + Shift + Z` | 重做 |
| `Ctrl + D` | 复制选中的东西 |
| `Ctrl + S` | 打开模板管理器 |
| `ESC` | 取消当前操作 / 关闭弹窗 |

## 3. T教练 AI — 智能战术助手

T教练 是你专属的 AI 战术教练。问什么都能回答——阵容搭配、地图打法、技能使用、职业比赛分析。

### 3.1 怎么开始

在战术板页面顶部点击 **「T教练」** 按钮（紫色），进入对话界面。

> 如果你只是来聊天的，不一定要画战术图。T教练 自己知道所有特工、技能和地图。

### 3.2 可以问什么

直接打字问，就像跟人聊天一样。比如：

- 「亚海悬城进攻方推荐什么阵容？」
- 「婕提在裂变峡谷怎么玩？」
- 「KAY/O 的闪存过载怎么用？」
- 「源工重镇 B 点怎么守？」
- 「当前阵容有什么问题？怎么改进？」
- 「分析我现在的战术布局」

AI 会把棋盘上你放的特工和技能位置也考虑进去。所以**先摆好阵容再提问**，回答会更准。

### 3.3 四种模式

| 模式 | 适合场景 |
|:---|:---|
| ⚡ 快速模式 | 日常问答，快问快答 |
| ⚖️ 均衡模式 | 战术分析、阵容推荐 |
| 💭 推理模式 | 深入研究、极致推理 |
| 🧠 深度模式 | 复杂战术推演、综合决策 |

**免费套餐**每天可以用 5 次快速模式。升级后解锁全部四种，每种每天有独立的次数额度。

### 3.4 让 T教练 更聪明

对话页面左侧有一些开关和设置，让你控制 AI 知道多少信息：

| 设置 | 说明 |
|:---|:---|
| 🗺️ 基础信息 | 开启后 AI 会读取棋盘上的特工站位和技能范围 |
| 📊 数据分析 | 选择比赛数据后，AI 可以结合你的历史对局给出建议 |
| 🎮 对局信息 | 选择地图、补充我方/敌方阵容，AI 会针对这局分析 |

> 💡 **建议**：提问前先确认这些开关。比如问阵容就选好当前地图和两边阵容。

## 4. 社区广场 — 发现和分享

点击底部（移动端）或侧边（桌面端）的 **「社区」** 进入社区。这里有四块内容：

### 4.1 战术广场

浏览其他玩家发布的战术布防图。可以按地图筛选、按热度或时间排序。看到喜欢的就点进去——

- 查看完整的战术布局（包括技能位置、进攻路线等）。
- 点赞、收藏、评论。
- 点击「加载到战术板」可以把它复制到你自己的战术板上修改。

### 4.2 技能点位库

一个专门存放**技能瞄点**的百科全书。每条点位包含：

- 特工名 + 技能名
- 站位坐标和落点坐标
- 截图（站位图、瞄准图、效果图）
- 难度评级（1-5 星）
- 评论区（可以讨论和改进点位）

想学某个技能的固定瞄点？来这里搜就对了。

### 4.3 论坛讨论区

发帖交流！有七种分类：

- 战术讨论：分享你的战术心得
- 英雄攻略：写特工使用教程
- 地图分析：讨论地图打法和攻防策略
- 开黑组队：找人一起玩
- 赛事讨论：聊 VCT 等职业比赛
- 训练技巧：提升枪法和意识
- 其他：自由话题

每条帖子都有点赞和评论功能。

### 4.4 个人主页

每个人都有自己的主页，展示：

- 发布的战术、点位和帖子
- 关注数和粉丝数
- 收藏的战术和点位

别人的主页上可以点击「关注」来订阅他的内容。

## 5. 常见问题

### 战术板相关

**Q：我换了电脑或浏览器，之前的战术还在吗？**
A：战术数据默认保存在当前浏览器的存储中，不会跨设备同步。需要跨设备请使用「导出 JSON」保存文件，在新设备上「导入 JSON」恢复。

**Q：地图上的技能标记太多了看不清怎么办？**
A：可以使用右下角的 👁‍🗨 按钮一键显示/隐藏所有技能范围。或者点击单个技能图标来只看这一个的范围。

### T教练 相关

**Q：免费版能用多久？**
A：永久免费！免费套餐每天可以使用 5 次快速模式。想用更多功能和更深的模式可以升级套餐。

**Q：T教练的战术建议准确吗？**
A：T教练会参考 VCT 职业比赛数据、社区战术分享和你的棋盘布局来生成建议。但战术是主观的——最适合你的打法才最重要。

**Q：我可以用自己的 DeepSeek / OpenAI API Key 吗？**
A：可以。在 T教练 页面左侧找到「自备 API Key」，¥19.9/月解锁，填入你自己的 Key 就能不限制次数使用。

### 社区相关

**Q：发布战术或点位需要注意什么？**
A：标题要清晰、描述要实用。含有违规内容的发布会被系统拦截。

**Q：怎么删除自己发布的内容？**
A：进入详情页，点击删除按钮（确认后不可恢复）。
