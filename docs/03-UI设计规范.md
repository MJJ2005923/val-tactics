# 03 — UI 设计规范

## 色彩系统

| 用途 | 色值 | 变量名 |
|------|------|--------|
| 主背景 | `#0a0a0a` | `--color-bg-primary` |
| 面板背景 | `#111111` | `--color-bg-secondary` |
| 卡片/控件背景 | `#1a1a1a` | `--color-bg-tertiary` |
| 悬停态 | `#222222` | `--color-bg-hover` |
| 激活态 | `#2a2a2a` | `--color-bg-active` |
| 主边框 | `#2a2a2a` | `--color-border` |
| 亮边框 | `#333333` | `--color-border-light` |
| 主文字 | `#e8e8e8` | `--color-text-primary` |
| 次要文字 | `#888888` | `--color-text-secondary` |
| 弱化文字 | `#555555` | `--color-text-muted` |
| 强调色（红） | `#ff4655` | `--color-accent` |

## 字体

| 用途 | 字体 | 大小 | 粗细 |
|------|------|------|------|
| 页面标题 | PingFang SC / Microsoft YaHei | 16px | 700 |
| 面板标题 | 同上 | 13px | 600 |
| 正文 | 同上 | 14px | 400 |
| 小字/辅助 | 同上 | 12px | 400 |
| 英雄名称 | 同上 | 14px | 500 |

## 间距

- 基础单位：4px
- 元素内间距：8px / 12px / 16px
- 组件间间距：16px / 24px
- 页面边距：16px

## 圆角

- 小（按钮/标签）：4px
- 中（卡片/面板）：6px
- 大（弹窗）：10px

## 交互规范

| 交互 | 反馈 |
|------|------|
| 按钮悬停 | 背景变亮 0.1s |
| 拖拽开始 | 图标放大 1.1x + 阴影 |
| 拖拽悬停 | 目标区域高亮边框 |
| 拖拽放下 | 标记从中心弹出动画 0.3s |
| 标记选中 | 边框高亮为强调色 |
| 弹窗打开 | 背景遮罩 0.2s fade-in |
| 时间轴折叠 | 高度过渡 0.3s |
