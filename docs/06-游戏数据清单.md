## 技能数据模板

每个英雄填写 4 个技能：

```typescript
{
  id: "sage-barrier-orb",
  name: "冰墙",
  nameEn: "Barrier Orb",
  key: "Q",
  type: "control",
  icon: "/images/abilities/sage/barrier-orb.png",
  description: "装备一个冰墙。按射击键在面前放置一堵冰墙...",
  usage: "按Q键装备，鼠标左键放置，可旋转方向"
}
```
